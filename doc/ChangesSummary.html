Summary of Argyll CMS Changes since last release

[V0.60 -> V0.70 Beta]


Apart from numerous bug fixes and many minor feature additions and improvements, the main changes to this version compared to the previous one are:
  • Support for the Eye-One Display and Eye-One Pro instruments.
  • Support for the DTP20 "Pulse" Instrument
  • Support for the Colorimètre HCFR instrument.
  • Support for the DTP22/Digital Swatchbook
  • Serial port instrument types are now automatically recognized, just like USB instruments. You still have to specify the serial port though.
As usual, a more detailed description of all changes is in the log.txt file that accompanies the source code.
