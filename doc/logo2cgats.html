profile/logo2cgats

Summary

Convert Gretag/Logo Profile Maker format test chart results into Argyll .ti3 CGATS format. 

Usage Summary

logo2gcats [-v] [-l limit] [devfile] infile [specfile] outfile

-l limit        set ink limit, 0 - 400%
[devfile]     Device CMYK target .txt file
infile          Input CIE, Spectral or Device & Spectral.txt file.
[specfile]    Input Spectral .txt file
outfile         Name for output.ti3 file

Usage Details and Discussion

logo2CGATS takes the Gretag/Logo test chart results, and converts them into Argyll .ti3 CGATS files.
It is quite common to find profile test chart data from various standards bodies and industry organizations, in this format, so it is useful to be able to convert them for use with Argyll. ICC profiles created using Gretag Profile Maker also commonly contain the test chart results embedded in the profile.

A variety of different packaging of Gretag/Logo data can be accepted:

1 source files, consisting of a combined device value and CIE and/or spectral values.
2 source files, consisting of  a file containing the device values, and a file containing the CIE and/or spectral values.
2 source files, consisting of a file containing the device values and the CIE values, and a file containing the spectral values.
3 source files, consisting of a file containing the device values, a file containing the CIE values, and a file containing the spectral values.

The Gretag/Logo test chart results format seem to change with each minor release of Profile Maker, so this utility may not work in all cases.
