spectro/synthcal

Summary

Create linear display calibration file.

Usage

synthcal [options] basename
 -s s1,s2,s3,    Set non-linear curve scale (default 1.0)
 -p p1,p2,p3,    Set non-linear curve powers (default 1.0)
 basename                Base file name for outut.

Comments

This is the utility creates a display calibration (.cal) file that has a linear table for each channel. This is useful in setting up a display for evaluation, testing, or to recover a display that has a strange set of Video LUTs (RAMDAC tables) loaded.

Optionaly it can be used to create a non-linear calibration file, useful for diagnostics. Each of the Red, Greed and Blue curves can be given a gamma (power curve) shape, and scaled to a maximum other than 1.0.
