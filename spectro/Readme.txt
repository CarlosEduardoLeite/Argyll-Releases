This directory containts the routines to read a print
test chart with an Xrite DTP51, DTP41, or a film recorder
test chart with a Gretag Spectrolino, and create an
appropriate CCGATS file containing the raw printer
calibration information.

printread.exe	Is used to read the print test chart and create
			    the chart readings file.

filmread.exe	Is used to read the film test chart and create
			    the chart readings file.
